### TrueCrypt Introduction

TrueCrypt is a free, open-source, portable virtual encrypted disk software. It allows you to create virtual encrypted disks on your hard drive without having to generate a physical file structure. Users can access these disks like any regular drive letter. All files on these virtual disks are automatically encrypted on-the-fly, requiring a password for access.
